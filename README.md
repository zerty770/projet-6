## Installation backend

```
à partir du dossier cloné :

"cd ./back/"
"mkdir images"
"npm install"
```

### Démarrage backend

```
"npm run dev"
```

## Installation frontend

```
à partir du dossier cloné (à la racine du projet) :

"npm install"
```

### Démarrage frontend

```
"ng serve"
```
